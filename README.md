# message_management
It is a web app which provides an platform to manage record of different kinds of programming messages(ERROR, INFO, etc) and their details.

Technologies used are:<br />
01 __node.js__<br />
02 __express.js__<br />
03 __mongoDB__<br />
04 __ejs__<br />
05 __css__<br />
06 __bootstrap__<br />
07 __semantic ui__
